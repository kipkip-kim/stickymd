# Sticky MD 개선 목록 (2026-03-13~)

## 버그 (우선)
1. 롤업하면 내용 삭제됨
2. 제목표시줄 아이콘 너무 커서 클릭/드래그 시 아이콘이 클릭됨
3. 트레이 아이콘이 파란 박스 — resources/icon.png 연결 안 됨

## UX 개선 (빠른 수정)
4. 새 메모 기본 크기 가로/세로 70% 증가
5. 새 메모가 기존 메모 왼쪽에 생성 (핀+확대 메모에서 새 메모가 가려지는 문제)
6. 메모 목록에서 해당 메모 윈도우를 최상단으로 올리는 버튼

## UX 개선 (설계 필요)
7. 색별 메모 목록 그룹핑 및 자동 정렬
8. 메모창 마그네틱 기능 (창끼리 가까이 가면 스냅)

=====================================
  Sticky MD v1.0.0
  경량 마크다운 스티키 메모
=====================================

바탕화면에 붙이는 메모장입니다.
마크다운 문법을 지원하며 작성한 내용이 .md 파일로 자동 저장됩니다.


[ 주요 기능 ]

  메모 작성
  - 바탕화면 어디서나 메모를 만들고 자유롭게 배치
  - 마크다운(Markdown) 실시간 편집 (제목, 굵게, 기울임, 체크박스 등)
  - 슬래시 명령어(/) 입력으로 빠른 서식 적용
  - 토글(접기/펼치기) 블록 지원
  - 메모 내 검색 (Ctrl+F)

  메모 관리
  - 메모 목록에서 전체 메모 확인, 검색, 정렬
  - 메모 색상 변경 (7가지 기본 색상 + 사용자 지정)
  - 휴지통 (삭제 후 30일간 복원 가능)
  - .md / .txt 파일 가져오기 및 내보내기
  - 드래그 앤 드롭으로 파일 내용 가져오기

  편의 기능
  - 시스템 트레이 상주 (닫아도 백그라운드 유지)
  - 메모 접기/펼치기 (타이틀바만 표시)
  - 메모 항상 위에 고정 (핀)
  - 투명도 조절 (30~100%)
  - 글꼴 크기 조절 (Ctrl+마우스 휠)
  - 다크 모드 (시스템 설정 연동 / 수동 전환)
  - 알람 설정 (1회, 매일, 요일별, 기간별)

  설정
  - 글꼴 변경 (시스템 글꼴 목록에서 선택)
  - 자동 저장 간격 조절
  - 저장 경로 변경
  - Windows 시작 시 자동 실행
  - 전역 단축키 설정
  - 툴바 버튼 구성 변경
  - 타이틀바 크기 (소형/기본/대형)
  - 백업 및 복원


[ 사용 방법 ]

  시작하기
  1. 프로그램을 실행하면 새 메모가 하나 생성됩니다.
  2. 메모에 바로 내용을 입력하세요.
  3. 입력한 내용은 자동으로 저장됩니다.

  새 메모 만들기
  - 시스템 트레이 아이콘을 더블클릭
  - 트레이 아이콘 우클릭 > 새 메모
  - 메모 목록 창에서 + 버튼

  메모 관리
  - 트레이 아이콘 우클릭 > 관리자 창
  - 메모 목록에서 클릭하면 해당 메모가 열립니다.

  서식 입력
  - 하단 툴바의 버튼 사용 (굵게, 밑줄, 체크박스 등)
  - 슬래시(/)를 입력하면 명령어 목록이 나타납니다.
  - 마크다운 문법을 직접 입력해도 됩니다.
    예) # 제목  **굵게**  - [ ] 할일

  단축키
  - Ctrl+B        굵게
  - Ctrl+I        기울임
  - Ctrl+U        밑줄
  - Ctrl+Z        실행 취소
  - Ctrl+Y        다시 실행
  - Ctrl+F        메모 내 검색
  - Ctrl+마우스휠  글꼴 크기 조절


[ 데이터 저장 위치 ]

  메모 파일:  내 문서\StickyMemo\  (*.md)
  앱 설정:    %AppData%\StickyMemo\  (settings.json, state.json)

  메모는 일반 마크다운(.md) 파일로 저장되므로
  다른 텍스트 편집기에서도 열어볼 수 있습니다.


[ 시스템 요구사항 ]

  - Windows 10 이상
  - 약 200MB 디스크 공간


[ 알려진 사항 ]

  - 코드 서명이 없어 설치 시 Windows SmartScreen 경고가
    나타날 수 있습니다. "추가 정보" > "실행"을 클릭하세요.
  - 프로그램을 제거해도 메모 데이터(내 문서\StickyMemo)는
    삭제되지 않습니다. 필요 시 직접 삭제하세요.
